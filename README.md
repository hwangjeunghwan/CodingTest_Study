## 스터디 목적

## 스터디 방법
- 매주 인프런의 코딩테스트 강의문제를 4문제 풀고 깃으로 커밋한다.
- 강의를 듣고 문제를 풀었다면 주석에 강의라는 표시를 넣어준다.

## 코드 리뷰 방법
- 깃으로 커밋한 상대의 코드를 들어가 분석해보고 공부한다.


## 진행 방법
- 본인의 이름에 해당하는 디렉토리를 각자 IDE에서 workspace로 잡는다.
- 각자 이름 폴더내에 섹션별 폴더를 만들어 코드를 커밋한다.
